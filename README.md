# account-book
> A simple account book built in React.js

## Requirements

- macOS
- node 12.0+

## Start

### Installation
```bash
  $ npm i
```

### Run it
```bash
  $ npm run dev
  ℹ ｢wds｣: Project is running at http://localhost:8080/
```

### Check it
Open your browser at http://127.0.0.1:8080/
